### Trie树(字典树)

```
- 概念
字典树，即Trie树，又称单词查找树或键树，是一种树形结构
- 应用
用于统计和排序大量的字符串（但不仅限于字符串），所以经常被搜索引擎系统用于文本词频统计

- 基本性质
1. 节点本身不存完整单词
2. 从根节点到某一节点，路径上经过的字符连接起来，为该节点对应的字符串
3. 每个节点的所有子节点路径代表的字符都不相同
```

> 二叉搜索树

```
- 概念
左子节点总是小于根节点，右子树总是大于根节点，适用于所有节点
- 前序
根左右
- 中序
左根右（二叉搜索树顺序遍历）
- 后序
右左根
```
